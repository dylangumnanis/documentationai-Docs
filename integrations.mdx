## Domain and Email Integrations

### Domain Setup

Follow these steps to connect a custom domain.

<Steps>
  <Step title="Purchase Domain" icon="shopping-bag">
    Acquire a domain from a registrar like GoDaddy or Namecheap.
  </Step>
  <Step title="Update DNS" icon="settings">
    Add these DNS records in your provider's panel:

````bash
# CNAME for www
www.example.com  CNAME  snowai.app

# A record for apex domain
example.com     A      192.0.2.1
````
  </Step>
  <Step title="Verify in Dashboard" icon="check-circle">
    Go to Snow AI dashboard > Domains > Verify. Propagation takes `<24h`.
  </Step>
</Steps>

### Email Setup

<Expandable title="Advanced Email Configuration" default-open="false">
Create business email like `info@yourdomain.com`.

1. Enable email forwarding in Snow AI settings.
2. Point MX records:

```
MX  Priority  Hostname
10  mx1.snowai.app
20  mx2.snowai.app
```
</Expandable>

## Webhooks for Custom Automations

Webhooks enable real-time notifications from Snow AI to your services. Use `POST` to `https://your-webhook-url.com/webhook`.

<Request show-lines="true">
```javascript
// JavaScript (Node.js)
const response = await fetch('https://api.example.com/v1/webhooks', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
  body: JSON.stringify({
    url: 'https://your-webhook-url.com/webhook',
    events: ['user.signup', 'order.created']
  })
});
```

```python
# Python
import requests
response = requests.post('https://api.example.com/v1/webhooks', 
  headers={'Authorization': 'Bearer YOUR_API_KEY'},
  json={'url': 'https://your-webhook-url.com/webhook', 'events': ['user.signup']}
)
```

```bash
# cURL
curl -X POST https://api.example.com/v1/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"url":"https://your-webhook-url.com/webhook","events":["user.signup"]}'
```
</Request>

**Sample Payload:**

```json
{
  "event": "user.signup",
  "data": {
    "userId": "usr_123",
    "email": "user@example.com"
  },
  "timestamp": "2024-01-15T10:30:00Z"
}
```
