## AI-Assisted Website Design

Build custom websites by describing your vision in plain English. Snow AI generates responsive designs, complete with navigation, pages, and styling.

### Quick Start Workflow

<Steps>
  <Step title="Describe Your Site" icon="message-circle">
    Enter a prompt like: "Create a modern portfolio site for a photographer with gallery, about, and contact pages."
  </Step>
  <Step title="Review & Customize" icon="edit">
    Preview the generated site and tweak elements via chat: "Make the header purple and add a booking button."
  </Step>
  <Step title="Deploy Instantly" icon="rocket">

</Steps>

## CRM and Booking Forms Integration

Integrate CRM and forms effortlessly using natural language. Snow AI handles the backend connections.

<Tabs>
  <Tab title="Booking Forms" icon="calendar">
    Prompt: "Add a booking calendar for consultations integrated with Google Calendar."

    ```javascript
    // Example API call after generation
    const booking = await fetch('https://api.snowai.app/forms/booking', {
      method: 'POST',
      headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
      body: JSON.stringify({ event: 'consultation', date: '2024-12-01' })
    });
    ```
  </Tab>
  <Tab title="CRM Sync" icon="database">
    Prompt: "Connect to HubSpot CRM for lead capture from contact forms."

    <ParamField path="leads" param-type="object" required="true">
      Customer data including name, email, and source.
    </ParamField>
  </Tab>
</Tabs>

<Callout kind="tip">
Test integrations in preview mode before going live to ensure data flows correctly.
</Callout>

## Content and Copy Management

Generate and manage content dynamically. AI creates SEO-friendly copy, blog posts, and product descriptions.

<Expandable title="Advanced Content Prompts" default-open="false">
Use specific prompts for best results:

````bash
# Generate blog post
snowai generate content --type blog --topic "AI in web design" --tone professional
````

This outputs optimized Markdown ready for your site.
</Expandable>
